## Complements of cfdi

Este EndPoint Obtiene un listado de código y leyenda de los complementos de CFDI aceptados.

### **URL**

```textmate
    GET /api/v1/complements/cfdi
```

### Headers

`Authorization: Bearer {token}`

`Accept: application/json`

### **URL Params**

```text
    None
```

### **Data Params**

```text
    None
```
